# 🧑‍💻 給家長的白話文教學：如何免費申請 AI 智慧大腦 (Gemini API Key)

這份說明書專門為「只會用 Word、平時不碰程式碼」的家長所寫。
跟著以下步驟做，大約只需要 **3 分鐘**，就能為「意圖偵測守衛」裝上聰明的 AI 大腦，讓它自動分辨孩子看的是「英文教學」還是「搞笑影片」。

---

## 💡 為什麼需要申請這個？
「API Key」就像是一把**專屬鑰匙**。
守衛程式需要這把鑰匙，才能免費連線到 Google 的 AI 伺服器，問它：「請問這個標題的影片，是學習還是娛樂？」。

別擔心，這完全是**免費的**，而且只需要操作一次！

---

## 🎯 步驟一：開啟申請網站

1. 打開你的瀏覽器（Chrome 或 Safari 都可以）。
2. 在網址列輸入並前往這個官方網址：
   👉 **https://aistudio.google.com/apikey**
3. 如果畫面要求登入，請輸入你平時使用的 **Google 帳號（Gmail）與密碼**。

---

## 🎯 步驟二：點擊「取得鑰匙」按鈕

1. 登入成功後，你會看到英文的 Google AI Studio 畫面（不用管上面的複雜英文）。
2. 請尋找畫面上方或左邊，一個藍色的按鈕寫著 **「Get API key」** 或 **「Create API key」**（建立 API 金鑰）。
   - 🔵 *[按鈕圖示：➕ Create API key]*
3. 大膽地**點擊它**！

---

## 🎯 步驟三：確認建立金鑰

1. 點擊後，畫面中央會跳出一個小視窗。
2. 點擊藍色的 **「Create API key in new project」**（在一個新專案中建立金鑰）。
3. 稍微等待幾秒鐘，Google 正在為你打造專屬鑰匙。

---

## 🎯 步驟四：複製專屬鑰匙

1. 成功後，畫面上會出現一長串像亂碼一樣的英文字母與數字（這就是你的 API Key）。
   - *長得像這樣：`AIzaSyB9xxxx_xxxxxxxxxxxxxxxxxx`*
2. 密碼旁邊通常會有一個 **「Copy」**（複製）按鈕，長得像兩個重疊的方形 📋。
3. 把滑鼠移過去**點擊它**（就等於你按了右鍵 → 複製）。
   - ⚠️ **注意：這串密碼就像提款卡密碼，請不要傳到網路上或給陌生人。**

---

## 🎯 步驟五：把鑰匙貼到守衛程式裡

現在，我們要把剛才複製的鑰匙，貼到我們的守衛程式裡交給它。

1. 在你的電腦上，找到你存放守衛程式的資料夾（`python project/guard`）。
2. 對著 `guard.py` 這個檔案，按滑鼠**右鍵**，選擇 **「打開檔案的應用程式」** → 選擇 **「文字編輯」**(TextEdit) 或任何文書軟體。
   *(不要點兩下執行它，我們要編輯它裡面的文字)*
3. 檔案打開後，稍微往下看（在很前面的大約第 28 行），找到這一段文字：
   ```python
   # ─── Gemini API 設定 ──────────
   # 填入你的 Gemini API Key（免費申請：https://aistudio.google.com/apikey）
   # 留空字串 "" 則改用關鍵字規則模式
   GEMINI_API_KEY = ""   # ← 貼上你的 API Key
   ```
4. 把滑鼠游標點在 `""` 這兩個引號**中間**。
5. 按下鍵盤上的 **Command + V**（或右鍵 → 貼上）。
6. 修改後，它應該會長得像這樣（引號裡面包著你剛才複製的那長串字母）：
   ```python
   GEMINI_API_KEY = "AIzaSyB9xxxx_xxxxxxxxxxxxxxxxxx"
   ```
7. 最後，點擊左上角功能表 **「檔案」→「儲存」**（或按鍵盤 Command + S）。
8. 關閉這個文字檔案。

---

## 🎉 完成！大功告成

你已經成功為守衛裝上 AI 大腦了！現在，你可以像平常一樣啟動守衛。
```bash
python3 guard.py
```

當孩子打開 YouTube 時，守衛會自動透過這把鑰匙，把影片標題傳給 Google 的 AI，並在閃電般的時間內聰明地判斷出「這是學習還是娛樂」！
