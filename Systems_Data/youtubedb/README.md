# 蕭博士 SoR 內容生產工廠 (SoR Content Factory)

這是基於 Science of Reading (SoR) 理論開發的自動化內容生產系統，旨在協助將影音素材轉化為深度的行銷文案與知識庫。

## 🚀 快速上手 (新電腦部署)

### 1. 環境需求
- **Python 3.10+**
- **FFmpeg**: 系統必須安裝 FFmpeg (用於音訊提取)。
  - Mac: `brew install ffmpeg`
  - Windows: [下載並設定環境變數](https://ffmpeg.org/download.html)

### 1. 懶人啟動 (推薦)
如果您在 **Mac** 上，直接點擊專案根目錄下的 **`啟動系統.command`** 即可。它會自動檢查環境、安裝必要的套件並開啟程式。

### 2. 手動安裝步驟 (進階)
3. **設定環境變數**:
   - 在 `core/` 資料夾中，找到 `.env.example` 並複製一份命名為 `.env`。
   - 填入您的 `OPENAI_API_KEY`、`GOOGLE_API_KEY` 與 `OPENROUTER_API_KEY`。

### 3. 啟動系統
在專案根目錄下執行：
```bash
streamlit run Home.py
```

## 📦 如何打包 (減輕體積)
目前 `data/downloaded_files/` 包含大量暫存音訊 (約 2GB)，打包時**不需要**包含它們。

在專案根目錄下執行此指令，可產出一個只有幾 MB 的 `youtubedb_slim.zip`：
```bash
zip -r youtubedb_slim.zip . -x "data/downloaded_files/*" "__pycache__/*" ".git/*" ".DS_Store"
```

### 必須保留的檔案：

- ✅ `requirements.txt`, `README.md`, `啟動系統.command` (啟動必備)
- ✅ `data/sources/` (已生成的完整逐字稿與策略文案)
- ✅ `data/dr_hsiao_db/` (向量搜尋資料庫)
- ✅ `data/videos.json` (影片目錄索引)
- `Home.py`: 系統入口首頁。
- `pages/`: 各項功能模組 (採收、搜尋、精煉、本地處理)。
- `core/`: 核心邏輯、Prompt 設定與工具。
- `data/`: 存放下載的音訊、生成的逐字稿與策略文案 (建議定期備份)。

## 🧠 核心功能
1. **YouTube 採收**: 自動下載並轉錄 YouTube 影片。
2. **文案精煉**: 將逐字稿轉化為「科學靈魂+比喻外殼+Q&A」的三段式深度文案。
3. **本地處理**: 批次處理本機影片，支援 AI 智能跳過片頭。
4. **知識庫搜尋**: 向量資料庫語意搜尋。

## 🛠️ 維護說明
- **修改 Prompt**: 到 `core/prompts.json` 修改文案生成的模板。
- **資料清理**: 若磁碟空間不足，可清理 `data/downloaded_files/` 下的轉檔暫存。
