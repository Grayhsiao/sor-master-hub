# 蕭博士 SoR - YouTube 知識庫系統 (乾淨版)

這是一套可以自動採收 YouTube 影片、生成逐字稿、並精煉出 SoR 策略文案，最後建立向量語意搜尋引擎的系統。

## ⚠️ 系統需求與安裝準備

在開始之前，請確認你的電腦已安裝以下基礎程式：
1. **Python (大於 3.9 版)**：[下載 Python](https://www.python.org/downloads/)
2. **FFmpeg**：用來處理音訊的大型核心元件。
   - **Mac 用戶**：開啟終端機 (Terminal) 輸入 `brew install ffmpeg`。
   - **Windows 用戶**：建議使用 [Scoop](https://scoop.sh/) 安裝，或手動下載佈署環境變數。

---

## 🚀 系統安裝步驟

### 1. 安裝所需套件
打開終端機 (Terminal / 命令提示字元)，切換到這個資料夾下，執行以下指令：
```bash
pip install -r requirements.txt
```

### 2. 設定 API Keys
本系統高度依賴 AI 運算，你需要自己準備 API 金鑰：
1. 將 `core/.env.example` 複製一份，並改名為 `core/.env`（注意前面有一個小數點）。
2. 用記事本打開 `core/.env`，填入你的 API Key。

```env
OPENAI_API_KEY="sk-你的OpenAI金鑰填這裡..."
GOOGLE_API_KEY="AIzaSy...你的Google金鑰填這裡..."
OPENROUTER_API_KEY="sk-or-v1-...你的OpenRouter金鑰填這裡..."
```

*(註：若只有其中一種，系統功能可能會受限，建議至少要有 OpenAI 才能進行高準度語音轉錄)*

---

## 🏃 如何啟動系統

每次要使用時，打開終端機，在這個資料夾下執行：
```bash
streamlit run Home.py
```
執行後，瀏覽器會自動跳轉，即可開始建立你自己的 YouTube 知識庫！
