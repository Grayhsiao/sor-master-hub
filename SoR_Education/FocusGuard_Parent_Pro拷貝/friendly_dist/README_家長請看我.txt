# 🛡️ Focus Guard Pro 家長使用說明書

親愛的家長您好：

感謝您使用 Focus Guard Pro 守護系統。本軟體旨在幫助您的孩子在電腦學習時保持專注，避免受干擾內容影響。

### 🚀 第一步：啟動守護
1. 請直接點擊「FocusGuardPro」圖示（帶有 Andy 娃娃的那個）。
2. 打開後，您會看到一組「遠端識別代碼」（例如：PRO8747），請記下它。
3. 此時「影子守護」已自動啟動，軟體會安靜地在背景運作。

### 📱 第二步：開始遙控
1. 用您的手機掃描或點擊 SoR Hub 的「進入家長控制台」。
2. 輸入剛才記下的「遠端識別代碼」。
3. 您可以隨時下達「計時鎖定」或「修改緊急密碼」。

### 💡 溫馨提醒
- 軟體具備「斷電記憶」，即便孩子重開機，之前設定的專注時間也會自動恢復。
- 如需緊急解除，請在地端輸入您的 PIN 碼（預設為 1234）。

祝您的孩子學習順利！
蕭博士 SoR 團隊 敬上
